IR Remote - Fichiers audio requis
===================================

Placez les fichiers .wav suivants dans ce dossier :

  - click.wav    (son de touche, ~0.1s, 44100Hz, mono)
  - success.wav  (son de succès, ~0.3s, 44100Hz, mono)
  - error.wav    (son d'erreur, ~0.3s, 44100Hz, mono)

Format attendu : PCM 16-bit, 44100 Hz, mono, WAV.

Vous pouvez générer ces fichiers avec Audacity (gratuit) ou tout éditeur audio.
